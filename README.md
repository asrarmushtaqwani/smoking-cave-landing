# Smoking Cave Cigar Lounge - Landing Page

A minimalist, elegant landing page for Smoking Cave Cigar Lounge in Highlands Ranch, Colorado.

## Features

- 🎨 Minimalist black dark theme design
- 📱 Fully responsive mobile-friendly layout
- 🗺️ Embedded Google Maps location
- 📧 Newsletter subscription form
- 🔗 Integration with all website pages
- ⚡ Fast, lightweight, and SEO-friendly

## Pages Included

- Homepage with hero section
- About Us - History and amenities
- Featured Brands - Caldwell, Foundation, JSK Cigars
- Location - Google Maps embed
- Newsletter signup
- Contact information and links

## Local Development

To run locally:

```bash
python3 -m http.server 3000
```

Then visit `http://localhost:3000`

## Deployment

This site is automatically deployed to Vercel whenever changes are pushed to the main branch on GitHub.
